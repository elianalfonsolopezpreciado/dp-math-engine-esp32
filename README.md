# dp-math-engine-esp32
Source code and experimental materials from the article "Dynamic Precision Math Engine for Linear Algebra and Trigonometry Acceleration on Xtensa LX6 Microcontrollers" (Elian A. Lopez Preciado, Mar 2026). It implements Q16.16 fixed-point arithmetic, a 16-iteration CORDIC kernel, a tiles-based matmul, and a FreeRTOS-compatible runtime precision-switching mechanism.
